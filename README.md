# Animated Side Navigation Menu

Projeto desenvolvido com HTML e CSS que apresenta um menu lateral animado. A navegação é aberta e fechada através de um botão interativo, utilizando apenas HTML e CSS, sem necessidade de JavaScript.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Font Awesome
- Checkbox Hack

## ✨ Funcionalidades

- Menu lateral animado
- Botão para abrir e fechar o menu
- Ícones personalizados com Font Awesome
- Links de navegação
- Ícones de redes sociais
- Animação suave com CSS
- Projeto sem JavaScript

## 📂 Estrutura do Projeto

```bash
Animated-Side-Navigation/
│
├── index.html
│
├── css/
│   └── style.css
│
└── img/
    └── icone.png
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.

## 📚 Conceitos Praticados

- Estruturação com HTML
- Menu lateral com CSS
- Uso de `input checkbox`
- Uso de `label` com `for`
- Ícones com Font Awesome
- Position Fixed
- Transitions
- Animações sem JavaScript
- Organização de layout

## 💡 Objetivo

Praticar a criação de menus laterais interativos usando apenas HTML e CSS, reforçando conceitos de posicionamento, transições e controle de estado com checkbox.
